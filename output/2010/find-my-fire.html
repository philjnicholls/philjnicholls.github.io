Find My Fire

Worldwide tracking of cigarette lighters

A personal project to track lighters around the world by using a short domain and unique code generated for each lighter.

Features

  • Mapbox integration to show locations logged on a world map
  • Find my location using browser location
  • Geolocate for finding locations without GPS
  • CSS rendering of passport stamps to simulate different colors depending on location
  • General design based on a passport
  • Social login
  • Specific user flows for landing pages to funnel users to log locations
